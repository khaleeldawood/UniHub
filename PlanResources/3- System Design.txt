#### **UI Elements:**

* Cards: Points, Badges, Contributions, Rankings
* Charts: Bar/Line/Pie (Recharts/Chart.js)
* Leaderboard table
* Responsive grid layout

#### **Dashboard Pages**

1. **User Dashboard (/dashboard)**

    * Add **Notifications panel/dropdown**:
        * Displays **all unread/read notifications**.
        * Filters by type (`LEVEL_UP`, `BADGE_EARNED`, `EVENT_UPDATE`, `BLOG_APPROVAL`, `SYSTEM_ALERT`).
    * Optional **pop-up/modal** for instant feedback on new badges/level promotion.
    * **Settings Page (/settings)**
        * Add option to **enable/disable notifications pop-ups**.
